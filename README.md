### <u>What you will get</u>

<h4><li>Theme:</li></h4>

### <u> How it works </u>

<p>1.  Install the extension</p>
<p>2.  Restart VSCode</p>
<p>3.  Select the 'DarkPixel'-theme under editor settings.</p>
<p>4.  You're done, enjoy the 'bright' side of dark.</p>
